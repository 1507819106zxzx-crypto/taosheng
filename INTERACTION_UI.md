# 互动界面与可破坏家具（v0.3）

## 1) 世界家具（桌/椅/床/柜/沙发/电器）

### 触发（靠近）
- 靠近可交互家具（桌/椅/床/柜/沙发/冰箱/电视/PC/马桶/洗手台/灯/开关）时，屏幕角落出现提示：`E 交互`。
- 家具上方出现像素气泡菜单（默认高亮“搬运”），避免到处点不到。

### 搬运模式（背着走）
- 进入：对准家具按 `E`（或在气泡里选“搬运”）。
- 状态效果：
  - 角色背着家具的真实贴图（不是方框）。
  - 移速降低，禁止冲刺（Shift 无效）。
  - 禁止开枪（避免“搬家还开枪”的离谱操作）。
- 操作：
  - `左键`：放下（落点显示半透明预览）。
  - `Shift + 左键`：投掷（当作一次攻击；命中/落地可造成破坏）。
  - `右键` / `Q` / `Esc`：取消搬运（回到原位）。
  - `R`：旋转 90°（规划项：可扩展为滚轮微调）。

### 破坏与掉落
- 家具都有耐久（按材质区分：木质更脆、金属/电器更硬）。
- 近战（`J`）或投掷命中会扣耐久；耐久归零时：
  - 木质家具掉落 `wood`
  - 金属/电器掉落 `scrap`

## 2) 武器互斥与显示

- 武器互斥：手上拿枪时不能近战；想用木棍/铁管，必须在背包装备近战，它会自动收起枪。
- 显示方式：枪和棍棒都在“缩放后的画面层”绘制，旋转像贴图一样稳定像素（参考 `Q:\\huigui\\xiaotou` 的做法）。

## 3) 交互气泡 UI（规划）

- 统一气泡结构：1 行主动作 + 1~3 行次动作（同一套输入规则）。
- 动作优先级建议：搬运 > 开门/拾取 > 使用（开关/电视） > 拆解（后续）。
