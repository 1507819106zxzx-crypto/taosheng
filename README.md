# 幸存者 Demo + 硬核生存原型（pygame 像素风）

- 内部像素画布：`480x270`（整数倍缩放，像素保持一致）
- 运行入口：`run_game.bat`
- 文档：`GDD.md` / `ROADMAP.md`

## 模式
- 硬核生存（原型）：无限分块地图 + 建筑搜刮 + 背包拾取 + 手枪射击 + 僵尸 + 末日房车/自行车（上/下车）
- 幸存者 Demo：角色/场景选择、升级三选一

## 启动
双击 `run_game.bat`（自动建 venv + 安装依赖）。

## 操作
通用：
- 移动：`WASD` / 方向键
- 返回：`Esc`

硬核生存（原型）：
- 拾取：`E`
- 背包：`Tab`（方向键/WASD 选格，`Enter` 装备枪，`Q` 丢弃）
- 射击：鼠标左键（需要先捡到并装备枪）
- 换弹：`R`
- 载具：`F` 上/下车（优先最近的：房车/自行车；房车有油耗）
- 房车驾驶：`W/S` 前进/倒车，`A/D` 转向（支持方向键）
- 进屋效果：进入建筑后屋顶半透明
- 冲刺：`Shift`

幸存者 Demo：
- 选择：方向键 + `Enter` / 鼠标左键
- 网格：右上角 `网格:开/关`
